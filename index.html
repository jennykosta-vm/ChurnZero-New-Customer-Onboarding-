Account = Active, Master Path = Won/Intro Success, CX Manager is not empty, Launch Call Date is empty
Segment buildout rule: All segments include "Account Journey → Journey = New Customer Onboarding" and "Status → Has complete = false" to trigger milestone actions.
How to read this flow: The journey starts when an account enters the trigger segment (purple). Two initial plays (red) fire immediately. Then the journey moves through five sequential milestones (blue). Within each milestone: green = achievements (progress markers), red = plays (automated comms), yellow = CSM tasks (manual items). Each milestone activates only when all steps in the prior milestone complete. Pink dashed tags show Master Path field updates.
